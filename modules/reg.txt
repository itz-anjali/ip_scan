$	    Dollar	 "Line yahan khatam honi chahiye."

import re

re.match() - Darwaze pe check karna Ye check karta hai ki kya
 line ki shuruwat us pattern se ho rahi hai?

Example: Kya sentence "Hello" se start hua?

2. re.search() - Poore ghar mein dhoondna Ye poori line mein kahin 
bhi wo cheez dhoondta hai. Par ye sirf pehla wala dhoond ke 
ruk jata hai.

Example: Is line mein kahin bhi "Apple" likha hai kya?

3. re.findall() - Sab kuch dhoondna (Sabse Kaam ki cheez) Ye poore 
text mein jitni baar wo cheez aayi hai, sabko dhoond ke list bana 
deta hai.

-------------------------------------------------------
import re

# Hamara Text
text = "Mera naam Rahul hai aur mera number 9876543210 hai. Mere dost ka number 1234567890 hai."

# Pattern (Logic samjho):
# \d ka matlab number.
# {10} ka matlab hamein lagataar 10 numbers chahiye.
pattern = r"\d{10}" 


result = re.findall(pattern, text)

print("Mujhe ye numbers mile:", result)

re.sub() (Replacement)

import re

text = "Call me at 9876543210"
# Jahan bhi digit (\d) mile, use "SECRET" se badal do
new_text = re.sub(r"\d+", "SECRET", text)

print(new_text)
